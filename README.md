## 🚀 Deployment Instructions

### Local Development
```bash
cd frontend-battle
pnpm install
pnpm run dev
```

### Production Build
```bash
pnpm run build
```

### Deploy to Vercel
1. Upload the `frontend-battle` folder to Vercel
2. Set framework preset to "Vite"
3. Build command: `pnpm run build`
4. Output directory: `dist`
5. Deploy!
